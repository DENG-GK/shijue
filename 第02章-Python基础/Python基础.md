    content = f.read()

## 🚨 常见问题

### Q1: Python 2 和 Python 3 有什么区别？
**A:** 主要区别：
- print 语句：Python 2 用 `print "hello"`，Python 3 用 `print("hello")`
- 整数除法：Python 2 中 `5/2=2`，Python 3 中 `5/2=2.5`
- 字符串：Python 3 默认使用 Unicode
- **建议：直接学习 Python 3，不要学习 Python 2**

### Q2: 缩进错误怎么办？
**A:** Python 使用缩进表示代码块：
- 统一使用4个空格（不要用Tab）
- VSCode 可以自动转换
- 同一代码块缩进必须一致

### Q3: 如何调试Python代码？
**A:** 多种调试方法：
1. 使用 `print()` 输出变量值
2. VSCode 断点调试（F5）
3. 使用 `pdb` 调试器
4. 使用 `logging` 模块记录日志

### Q4: 列表和元组有什么区别？
**A:**
- **列表（list）**：可变，用方括号 `[]`，可以修改元素
- **元组（tuple）**：不可变，用圆括号 `()`，不能修改元素
- 元组性能更好，适合存储不变的数据
