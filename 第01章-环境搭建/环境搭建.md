## 🛠️ 快速环境搭建指南

如果你已经熟悉环境搭建，可以按照以下步骤快速完成：

### Windows 快速搭建

```bash
# 1. 安装 VSCode
# 访问 https://code.visualstudio.com/ 下载安装

# 2. 安装 Python 3.11
# 访问 https://www.python.org/downloads/ 下载安装
# ⚠️ 勾选 "Add Python to PATH"

# 3. 创建项目目录
mkdir E:\学习用\cv_project
cd E:\学习用\cv_project

# 4. 创建虚拟环境
python -m venv venv

# 5. 激活虚拟环境
.\venv\Scripts\Activate.ps1

# 6. 升级 pip
python -m pip install --upgrade pip

# 7. 安装核心库
pip install numpy opencv-contrib-python matplotlib Pillow -i https://pypi.tuna.tsinghua.edu.cn/simple

# 8. 验证安装
python -c "import cv2; import numpy; print('✅ 环境搭建成功！')"
```

### macOS/Linux 快速搭建

```bash
# 1. 安装 Python（如果未安装）
# macOS: brew install python@3.11
# Ubuntu: sudo apt install python3.11

# 2. 创建项目目录
mkdir -p ~/cv_project
cd ~/cv_project

# 3. 创建虚拟环境
python3 -m venv venv

# 4. 激活虚拟环境
source venv/bin/activate

# 5. 安装核心库
pip install numpy opencv-contrib-python matplotlib Pillow -i https://pypi.tuna.tsinghua.edu.cn/simple

# 6. 验证安装
python -c "import cv2; import numpy; print('✅ 环境搭建成功！')"
```

## 🚨 常见问题

### Q1: VSCode 终端无法激活虚拟环境？
**A:** Windows PowerShell 可能禁止运行脚本，以管理员身份运行：
```powershell
Set-ExecutionPolicy RemoteSigned
```

### Q2: pip 安装速度很慢？
**A:** 配置国内镜像源（清华、阿里云）：
```bash
pip install package_name -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q3: import cv2 报错？
**A:** 检查：
1. 虚拟环境是否正确激活
2. OpenCV 是否安装成功：`pip list | grep opencv`
3. Python 版本是否兼容

### Q4: DLL 加载失败（Windows）？
**A:** 安装 Visual C++ Redistributable：
https://aka.ms/vs/17/release/vc_redist.x64.exe
